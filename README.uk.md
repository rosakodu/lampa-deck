Плагін для Decky Loader на Steam Deck, який запускає повністю локальний екземпляр каталогу та плеєра **Lampa**, TorrServer-gst та кастомний транскодер **WebM VP8 + Opus** для безшовного відтворення торрентів у вбудованому плеєрі Lampa в Steam Big Picture (Ігровому режимі).

![Screenshot](assets/screenshot.png)

## 📋 Можливості

- **Локальний Lampa**: Роздає статичні файли Lampa на порту 8300.
- **Вбудований TorrServer**: Автоматично завантажує стабільний TorrServer-gst для локального стрімінгу торрентів на порту 8090.
- **Транскодування WebM VP8 + Opus**: Фоновий рушій транскодування на базі FFmpeg перетворює непідтримувані у SteamOS формати відео (HEVC/H.265, AV1, H.264) та аудіо (AC-3, DTS, AAC) у повністю вільний формат WebM (VP8 + Opus) на льоту. Це дозволяє відтворювати будь-які фільми прямо у вбудованому плеєрі Lampa без помилок звуку та зображення.
- **Продолжити перегляд**: Кнопка для запуску Lampa на останній відкритій сторінці/вкладці.
- **Панель керування**: Швидкий перезапуск TorrServer та кнопка очищення кешу (очищає базу даних TorrServer для звільнення місця).
- **Локалізація**: Повний переклад інтерфейсу на англійську, російську, українську, китайську (спрощену та традиційну), арабську, перську та турецьку мови.
- **Чисте встановлення**: Плагін постачається без попередньо встановлених плагінів, парсерів та сторонніх ресурсів. Налаштовуйте все під себе.

## 📦 Рекомендації з налаштування

Lampa вимагає ручного додавання джерел контенту. Ми рекомендуємо використовувати такі ресурси:

- **Адреса TorrServer**: Уже налаштована в Lampa за замовчуванням як `http://127.0.0.1:8090`.
- **Пошук по торрентах**: Рекомендуємо використовувати популярний парсер [JacRed](https://jacred.ru/) (`https://jacred.ru/`).
- **Корисні плагіни**:
  - **TMDB / Онлайн перегляд**: Репозиторій плагінів [nb557 Lampa Plugins](https://github.com/nb557/lampa-plugins) or плагін `https://plugin.rootu.top/tmdb.js`.
  - **Синхронізація CUB**: Сервіс [CUB.red](http://cub.red/) для зберігання закладок.
  - **Плагін пошуку Jackett**: Доступний через спільноту [Bylampa](https://github.com/bylampa/bylampa.github.io).

## 📥 Встановлення

1. Завантажте ZIP-архів (`lampa-deck.zip`) зі сторінки релізів.
2. Скопіюйте ZIP-архів на ваш Steam Deck.
3. Увімкніть **Режим розробника** в системних налаштуваннях Steam Deck.
4. У налаштуваннях Decky Loader перейдіть до розділу розробника, активуйте **Developer mode** та виберіть **Install plugin from file**.
